《星际争霸1》即为 《星际争霸》，加上数字 1 主要用于区别《星际争霸2》  
《星际争霸》自 [1.18](https://news.blizzard.com/en-gb/starcraft/20674424/starcraft-brood-war-patch-1-18-patch-notes) 版开始可免费下载游玩

《星际争霸：重制版》也属于《星际争霸1》
